# EasyHub Dashboard

Visual command center for EasyHub/OpenClaw.

## Features

- **Channel Status** - See connected channels at a glance (Telegram, WhatsApp, Discord, etc.)
- **Activity Feed** - Recent actions and events
- **Sessions List** - Active AI sessions and their status
- **System Health** - Gateway status, uptime, current model

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Stack

- **Svelte** - Reactive UI framework
- **Vite** - Fast build tool
- **CSS** - Custom dark theme, no external dependencies
