- 우선순위 추천: `install/*`, `channels/*`, `gateway/*`
- 용어 예시: `Gateway(게이트웨이)`, `pairing(페어링)`, `workspace(워크스페이스)`
- 기여 시작점: [Contributing](/help/submitting-a-pr)
