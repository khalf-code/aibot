# `zoidbergbot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
zoidbergbot pairing list whatsapp
zoidbergbot pairing approve whatsapp <code> --notify
```
